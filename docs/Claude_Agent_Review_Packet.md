# Claude / Agent Review Packet - OpenGoldSDR

## Purpose

Review OpenGoldSDR as a realistic open-source research project, not as hype.

The project idea is to test whether low-cost SDR, electromagnetic sensing, public geology data, GPS-tagged field notes, and community review can help narrow down where to look for gold-bearing ground.

## Important framing

Do **not** oversell this as "Wi-Fi finds gold."

Gold is hard to detect directly through soil with consumer radio hardware. A better research framing is:

- Detecting anomalies
- Reading soil moisture/conductivity changes
- Comparing mineralization signals
- Mapping geology and historical mine data
- Looking for quartz, old workings, terrain clues, and public-data patterns
- Designing repeatable field tests
