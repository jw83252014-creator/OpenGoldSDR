# Visual Instructions - OpenGoldSDR Hero Image

## Goal

Use a strong generated image as the repo/X hero visual.

The image should immediately communicate:

- Open-source technical project
- SDR / EM / signal analysis
- Field prospecting
- Georgia Gold Belt / geology map context
- Community review / contribution
- Prototype workbench energy

## Recommended overlay text

OpenGoldSDR
Open-source SDR / EM prospecting research
Can low-cost radio hardware + public geology data help narrow down where to look?
github.com/jw83252014-creator/OpenGoldSDR

## Visual direction

Use the Grok-generated image that already looks like:

- Prototype/workbench scene
- OpenGoldSDR screens
- Signal-analysis displays
- Community contribution panels
- Georgia Gold Belt mapping
- Blue/cyan technical lighting

## Avoid

- Do not imply guaranteed gold detection
- Do not show "Wi-Fi beams finding gold"
- Do not make it look like a finished commercial product
- Do not make fake scientific claims in the graphic

## Better image caption

OpenGoldSDR is not claiming magic gold detection. It is an open-source attempt to test whether low-cost sensing, signal processing, public geology data, and field logs can help narrow prospecting decisions.
